# Netriva Collection - Project Context

## Project Overview

Netriva Collection is a full-stack e-commerce platform for selling Pattu Sarees, Silk Sarees, Cotton Sarees, Bridal Collections, and other fashion products online.

The goal is to build an industry-level scalable e-commerce application similar to Amazon, Flipkart, and Myntra, specifically tailored for Netriva Collection.

## Technology Stack

### Frontend

* Next.js
* React.js
* Tailwind CSS
* Axios
* Context API / Redux Toolkit

### Backend

* Django
* Django REST Framework (DRF)
* JWT Authentication

### Database

* PostgreSQL (Production)
* SQLite (Development)

### Media Storage

* Cloudinary

### Deployment

* Frontend: Vercel
* Backend: Render
* Database: PostgreSQL
